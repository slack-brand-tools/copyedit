Quick reference

Headlines

  • Sentence case for all headlines
  • No period for single sentence or fragment
  • Period if two or more sentences
  • Subheads: sentence case with a period

Punctuation

  • Always use the Oxford/serial comma
  • Use em dash (—) with spaces, not --
  • Use "a.m." and "p.m." (lowercase, periods)
  • No ordinals in dates (April 15, not 15th)

Product names

  • Capitalize: Workflow Builder, Slack Connect, Slack AI, Do Not Disturb
  • Lowercase: huddles, channels, clips, canvases, lists, shortcuts
  • Never say "Slack Canvas" — use "canvases"
  • Einstein is retired — use "Trust Layer"

Spelling preferences

  • "okay" not OK or O.K.
  • "email" not e-mail
  • "e-book" not ebook
  • "canceled" not cancelled
  • "advisor" not adviser
  • "toward" not towards
  • "gray" not grey
  • "healthcare" (one word)

Common jargon to avoid

  • leverage → use
  • empower → help
  • utilize → use
  • streamline → simplify
  • enable → help, turn on
  • bandwidth → time, capacity
  • deep dive → analysis
  • ecosystem → environment

Style hierarchy

  • 1. Salesforce style guide
  • 2. AP Style
  • 3. Merriam-Webster

Salesforce overrides AP; AP fills gaps; M-W is the default for spelling.
